BRAIN AI · Hackathon Report

Unified Asset & Operations Brain

An AI-powered industrial knowledge intelligence platform for grounded RAG, knowledge graph exploration, root cause analysis, and compliance checks.

Node.js Express Grounded RAG Knowledge Graph RCA Compliance

Project Goal

Turn maintenance manuals, work orders, inspection records, and incident notes into a single dashboard that helps plant teams answer questions faster and reason more safely.

Why It Matters

Industrial teams often lose time searching across fragmented documents. BRAIN AI centralizes evidence, surfaces relationships, and keeps the assistant grounded in the actual corpus.

System Architecture

Frontend Modules

  • Overview dashboard for system status and quick actions.
  • AI Copilot for grounded Q&A with citations.
  • Knowledge graph explorer for entity relationships.
  • RCA workbench for 5-Whys and fishbone output.
  • Compliance audit screen for OISD-style checks.

Backend Modules

  • Express server with static UI hosting.
  • Copilot route with refusal gating and retrieval.
  • RCA route with safe fallback scenarios.
  • Compliance route with simulated plant state.
  • Knowledge graph rebuild route and cached graph data.

API Reference

Core Routes

  • POST /api/test-llm - validates model connectivity.
  • POST /api/copilot - grounded Q&A over the corpus.
  • POST /api/rca - incident analysis and root cause generation.
  • GET /api/compliance - compliance matrix and violation details.
  • POST /api/compliance/resolve - updates simulated state and reruns audit.

Data Sources

  • PMP-302 manual and work orders.
  • B-401 SOP and work orders.
  • C-102 inspection record.
  • OISD-STD-189 safety standard.
  • Near-miss incident report.
